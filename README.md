## What The Signal

BSUIR lab project on DSP. Cool libs: [blueprint](https://blueprintjs.com),
[highcharts](https://highcharts.com/), `math.js`, `dsp.js`, `fft-windowing`,
`Electron`, `react`, `redux`, `reselect`, `prettier`, `standard.js`, blablabla.
This stack is great!
